Wordle

        Rules:
        - You are provided with 5 guess attempts
        - You have to enter 5-letter words only
        - After each guess
            - If the letter is correct and at correct position, then the feedback is provided of green colour
            - If the letter is correct but at wrong position, then the feedback is provided of yellow colour
        - In the end
            - Win if the word is correctly guessed
            - Lose if the attempts are finished
